# Poem Remix

## When to Use
When a user wants to transform an existing poem — changing its form, style, tone, era, or voice while preserving its core meaning.

## Instructions

### Remix Types

#### Form Shift
Convert between poetic forms:
- Sonnet → Haiku sequence (distill to essentials)
- Free verse → Villanelle (find the repeating truth)
- Limerick → Sonnet (elevate the humor)
- Any form → Any form

#### Style Shift
Rewrite in a different poetic movement or era:
- Romantic → Modernist (lush → spare)
- Beat → Victorian (spontaneous → formal)
- Confessional → Imagist (interior → exterior)

#### Voice Shift
Rewrite as if by a different type of speaker:
- Child's perspective → Elder's perspective
- First person → Third person
- Whisper → Shout (intimate → declarative)

#### Medium Shift
Adapt for a different performance context:
- Page poem → Spoken word (add rhythm, repetition, audience address)
- Spoken word → Page poem (tighten, trust the reader)
- Poem → Song lyrics (add chorus, hook)

### Process
1. **Identify the core** — What is the poem *really* about? Strip away form to find the essential meaning.
2. **Choose the target** — What form/style/voice will serve this meaning differently?
3. **Rebuild, don't paste** — A remix is a new poem inspired by the old one, not a search-and-replace.
4. **Present both** — Show original and remix side by side so the user can see the transformation.
5. **Explain the choices** — Briefly note what was gained and what was traded in the remix.

### Key Principles
- A good remix reveals something new about the original's meaning
- Honor the source material — transformation, not destruction
- Some losses are inevitable — name them honestly
- The best remixes make the reader see both poems differently
