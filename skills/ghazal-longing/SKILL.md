# Ghazal

## When to Use
When the user asks for a ghazal, or when the topic involves love, longing, separation, divine yearning, or paradox.

## Instructions

### Structure
- **Couplets (sher)**: 5–15 self-contained couplets
- **Radif**: A repeating word or phrase at the end of both lines in the first couplet (matla), then at the end of the second line of every subsequent couplet
- **Qafia**: A rhyming word that precedes the radif
- **Maqta**: The final couplet traditionally includes the poet's name (takhallus) — use the agent's name or the user's name if given

### Pattern
```
____ qafia radif
____ qafia radif     ← matla (opening couplet, both lines end with pattern)

____
____ qafia radif     ← each subsequent couplet: only 2nd line repeats

____
____ qafia radif

____
____ qafia radif     ← maqta (final couplet, may include poet's name)
```

### Writing Process
1. **Choose the radif** — A word or short phrase that can bear repetition ("of rain", "again", "the door")
2. **Write the matla** — The opening couplet sets the tone and establishes the pattern
3. **Each couplet is a world** — Every sher should stand alone as a complete thought or image
4. **Vary the mood** — Move between yearning, defiance, ecstasy, grief, wit across couplets
5. **The maqta** — Sign off with self-reference or a final turn

### Key Principles
- Each couplet is **autonomous** — a ghazal is not a narrative; it's a constellation
- The radif creates unity through sound, not through story
- Embrace contradiction — a ghazal can be joyful in one couplet and devastated in the next
- The tradition spans Rumi, Hafiz, Ghalib, Agha Shahid Ali — draw on this depth
- In English ghazals, the radif and qafia pattern is more important than strict meter
