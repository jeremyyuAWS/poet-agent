# Haiku & Japanese Short Forms

## When to Use
When the user asks for haiku, tanka, senryu, renga, or any minimalist short-form poem.

## Instructions

### Haiku
- **Structure**: 3 lines — 5/7/5 syllables
- **Subject**: Nature, seasons, moments of awareness
- **Must include**: A kigo (seasonal reference) and a kireji (cutting word — a pause or juxtaposition)
- **Spirit**: Capture a single moment with clarity. Less is more. Suggest, don't explain.

Example structure:
```
[image from nature]     — 5 syllables
[expansion or shift]    — 7 syllables
[surprise/insight]      — 5 syllables
```

### Senryu
- Same 5/7/5 structure as haiku
- **Subject**: Human nature, humor, irony, relationships
- No seasonal reference required
- More witty and observational than haiku

### Tanka
- **Structure**: 5 lines — 5/7/5/7/7 syllables
- First 3 lines (kami-no-ku) work like a haiku
- Last 2 lines (shimo-no-ku) add emotional reflection or response
- Best for: love, longing, personal emotion

### Haiku Sequence
- Multiple haiku linked by theme but each standing alone
- Use when the user wants to explore a topic from multiple angles
- 3-7 haiku is typical

### Key Principles
- **Show, don't tell** — present an image, not a feeling
- **Present tense** — haiku live in the now
- **No metaphor** — haiku are literal observations that resonate
- **Juxtaposition** — the best haiku place two images side by side and let the reader find the connection
